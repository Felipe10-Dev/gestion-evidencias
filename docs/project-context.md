# Sistema de Gestión de Evidencias Técnicas

## Problema

Muchas empresas que realizan trabajos técnicos necesitan registrar evidencias fotográficas de su trabajo.

Actualmente muchas empresas usan aplicaciones de mensajería como WhatsApp o carpetas desordenadas.

Esto genera problemas como:

- pérdida de evidencias
- desorden en las fotografías
- dificultad para consultar trabajos realizados
- falta de control sobre las actividades de los técnicos

## Solución

Desarrollar un sistema digital que permita registrar, organizar y consultar evidencias fotográficas de forma estructurada.

El sistema estará compuesto por:

- una aplicación móvil para técnicos
- un dashboard web para administradores
- un backend API
- una base de datos
- almacenamiento de imágenes en Google Drive

## Roles del sistema

### Administrador

El administrador puede usar el sistema desde la web o desde la aplicación móvil.

Puede:

- crear proyectos
- crear equipos
- visualizar evidencias
- eliminar evidencias
- gestionar usuarios

### Técnico

El técnico utiliza principalmente la aplicación móvil.

Puede:

- iniciar sesión
- ver proyectos
- crear equipos
- subir evidencias
