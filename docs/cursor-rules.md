# Reglas de desarrollo

El backend debe seguir arquitectura MVC.

Separar en carpetas:

controllers
models
routes
services
middlewares

Tecnologías del backend:

Node.js
Express
PostgreSQL
Sequelize

Frontend web:

Next.js
TailwindCSS

Aplicación móvil:

Flutter

Buenas prácticas:

- código modular
- manejo de errores
- funciones pequeñas
- separar lógica en services
